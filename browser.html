        Frequently Asked Questions

        Is Swivro Browser free forever?

        Yes :)

        Does Swivro Browser use Chromium? 

        Yes, Swivro does use Chromium, however, we did not fork Chromium directly. This web browser uses modern web technologies (ElectronJS) and was made possible thanks to Skye Browser. We chose to fork Skye Browser because it gives us a solid foundation to build off of, and is written in ElectronJS. Why do we want ElectronJS? If we forked Chromium directly (which is what ElectronJS uses), we would have to write in C++ which is more time-consuming and complex. Essentially, Electron makes things way easier. Without Electron, we couldn't offer a web browser. 

        What's the difference between Swivro Browser and Skye Browser? They look the same.

        To date (January 31st, 2022), we've made about 60 commits (updates/changes) to Swivro Browser in total (after forking Skye Browser). These commits include the addition of Startpage, Qwant, and Swisscows as default search engine options in the browser, updating the browser dependencies, various security enhancements, simplifying things, removing broken unneeded features, and more. Basically, Argon Browser is a clone of Skye Browser but provides new features faster, security/dependency updates faster, and a simpler experience. As Swivro Browser is still in its beta phase, some minor bugs/issues are expected.

        Is Swivro Browser open source software?

        Swivro Browser is free software and is of course 100% open source with the code being available on GitHub (swivro/browser-desktop). You're welcome to fork (copy/clone) Swivro Browser at any time, as long as you comply with the GNU General Public License v3.0 software license (which Swivro Browser is licensed under).

        Does Swivro Browser support browser extensions?

        Swivro Browser currently has limited/partial support for browser extensions from the Chrome Webstore. We usually don't recommend installing extensions anyways, as they make your fingerprint more unique, making it easier for websites & online services to uniquely identify you.

        Will Swivro Browser support Android/iOS?

        We have zero plans to release an iOS browser, however, we do plan to release Argon for Android some time in the future. We're not sure what browser engine we would use, but it would probably be Gecko. We also have plans to modify Swivro Browser for Desktop to use Gecko in the future.
