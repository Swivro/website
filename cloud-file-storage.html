Cloud File Storage

Find a safe home for your online data (photos, documents, etc) today by exploring privacy-respecting cloud file storage solutions, seen below.

Nextcloud

Decentralized

Open Source

E2EE

Nextcloud is a suite of client-server software for creating and using file hosting services. Being free and open-source software, anyone is allowed to install and operate it on their own private server devices.

Notes:
You should ensure that your Nextcloud Hosting Provider supports end-to-end encryption. If they don't you'll have to trust that they won't peak at your files.

Nextcloud is completely open source and is decentralized. We suggest self-hosting it, but you can also use some publicly-available Nextcloud instances.

Website
FileN

Centralized

E2EE

With one of Europe's highest privacy standards, FileN will keep your data safe. All of your files will be stored in Tier IV ISO 27001 certificated high security datacenters located in Germany, utilizing AES 256 bit end-to-end encryption.

Notes:
FileN is based & hosted in Germany. Their free plan offers 10GB of cloud file storage, along with no bandwidth/file traffic limits.

Website
Mega

Centralized

E2EE

MEGA is a cloud based service which can be used from all major devices and platforms from anywhere with Internet. By properly applying end-to-end encryption, MEGA achieves actual privacy by design.

Notes:
Mega is based in New Zealand. Additionally, as mentioned, MEGA offers full end-to-end encryption for all of its users. Using Nextcloud is recommended over Mega if you want full control over your files and data.

Website
Proton Drive

Centralized

E2EE

Currently in Beta, Proton Drive is a cloud file storage service, made by the popular encrypted email service, ProtonMail. Proton Drive utilizes full end-to-end encryption, ensuring that your security & privacy is at its greatest.

Notes:
As Proton Drive is in beta, they do not offer any desktop or mobile apps, meaning it is only accessible via a web browser, and only to paying users.
