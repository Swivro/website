Introducing Decentrasearch

Decentrasearch improves your privacy and anonymity by sending your search query to a randomly-picked SearX node in our database. This means every time you search, you connect to a different server. This, in numerous ways, makes it harder for malicious actors to build a profile on you. There is also no central search authority.

There are currently 34 SearX nodes in our database. Your search query will be sent to one of them (picked randomly).

Code & Database is all in your browser.

No server-side dynamic code. All in your browser.

Absolutely zero data is collected from you

An encrypted HTTPS/TLS connection is forced.

Frequently asked questions

  • How is this decentralized if it is only available here?
    Decentrasearch, is not a search engine itself. When you submit your search, the query is sent to a privacy-respecting open source search engine (a SearX node), randomly picked from our database. This means there is no central search authority, making it decentralized.
  • Do you log my search queries?
    No. All code for this project is ran in your browser, so you can see everything that is going on.
  • Are all of the search engines used for this safe?
    We only send your search query to an open source decentralized search engine (a public SearX node). We verify that the SearX node meets our minimum security and privacy requirements before adding it to our database. To see all of the search engines currently in our DB, you can view the source of this page, and scroll down until you see some JS code that stands out.
  • How is this more private than using a SearX search engine directly?
    Decentralized Search (the name of this project) improves your privacy and anonymity by sending your search query to a randomly-picked SearX instance in our database. This means every time you search, you connect to a different server. This primarily, in numerous ways, makes it harder for malicious actors to build a profile on you. Keep in mind that there is also no central search authority.
  • Some search results are different each time I search, why?
    Currently, we are still in the process of making Decentrasearch more stable to ensure that search results are very similar for each time you submit your search query. At this time, you may see slightly different search results each time you submit your search query. This is primarily due to each SearX node operator having different default server/search configuration.
What is a SearX Node?
×
SearX is an open source search engine written in Python. Anyone can host their own public instance of SearX, which anyone can use (we call those SearX Nodes). Our database consists of many SearX Nodes. When you submit a search query here, your query is sent to one of the SearX instances instances in our database, picked randomly. This improves your privacy and anonymity.
