The Tor Network

Tor is the most popular anonymity network in the world. We'll explain how it works, why people use it, and how to stay secure.

Is Tor truly safe? How does it make you anonymous? What are the risks?

We're here to answer all of those questions. Below, we'll discuss what exactly the Tor Network is, how Tor Servers function properly, and how to stay safe while using the Tor Browser; with all sections being broken up into tabs below, which you can quickly navigate between.

Tor, short for The Onion Router, is an anonymity network powered by the Tor Project and thousands of volunteers across the universe. The Tor Network does not make you 100% anonymous, as that is an impossibility, but it definitely boosts your anonymity. The easiest and most safe way to access websites via the Tor Network is to use the official Tor Browser from torproject.org. The Tor Browser should only be downloaded from the official website (torproject.org).
The way Tor works is, once you go to a normal website via the Tor Browser, your connection is passed to a Tor Guard Node, which is the first server your traffic goes to. It is then passed to one or more relay servers, where your traffic is relayed from one server to another, and then is finally passed to the Exit Node, which is the node that reaches out to the internet. The exit node fetches your requested content, then passes it back through the relay of servers, to finally reach your machine. Tor Nodes (exit/relay/guard nodes) are mostly ran by volunteers, and sadly the government. Please see the next tab to see a simplified explanation of different types of Tor Nodes.

Simplified Tor Node Explanation:

Guard Node: This is the first Tor Node that your Tor Browser connects to. This is the only Tor Node that you directly connect to and the only Tor Node that can actually see your home IP address.

Bridge Node: By default, you do not use a Bridge Node when using Tor. If your ISP blocks Tor Guard Nodes, and you are unable to use a VPN, then requesting Bridge Node IP's from Tor to utilize would be a good idea. Bridge Node IP Addresses are not publicly listed in a directory anywhere, so ISP's cannot block every Tor Bridge node.

Relay Node: These type of Tor Nodes are nodes that receive and pass traffic to other relay nodes. They relay your web traffic from server to server. Relay Nodes cannot see your IP address, and just see the IP address of other relay nodes (or exit/guard nodes) that they are passing traffic to or are receiving traffic from.

Exit Node: The final Tor Node that your web traffic/request reaches. The exit node is the node that actually fetches your requested content from the clearnet, then passes it back through all of the relay nodes and the guard node, to your machine. Exit nodes are not used when visiting a Tor Hidden Service aka Onion Site.

The Tor Browser sets a certain screen size, and useragent, so that all people using the Tor Browser have a common fingerprint, and no Tor users can be uniquely identified. You should not change any of those default settings, because by doing so, you will be de-anonymized and can be uniquely identified.
A large amount of ISP's block the Tor Network, and if this is the case with your ISP, you should request Tor Bridges. These type of Tor Nodes (Bridges) are essentially Guard Nodes that are not publicly listed in any online directory. You should NOT use a VPN while using the Tor Network, as a money trail is often attached. Using a VPN in conjunction with Tor in an attempt to not fall on the watchlist of your ISP or government would be almost pointless as Tor has almost 2 million daily users.
A more serious issue is when your Tor Circuit is using a Malicious Exit Node. Exit Nodes are the only Tor Nodes that actually reach out to the internet (clearnet) to fetch your requested web content. If the operator of the Exit Node that you are using has bad intentions, they may be able to inspect some of the web traffic sent by you, going through that Exit Node, which could de-anonymize you if you are entering sensitive information into the website you are accessing. The best way to avoid this is to never visit sites through Tor that are on the clearnet (ones ending in .com, net, org, etc), and only visit Tor Hidden Services a.k.a Onion Sites (sites that end with .onion).
This is because Tor Hidden Services do not utilize Exit Nodes, so it would be impossible for you to be de-anonymized due to a malicious Exit Node.

Great, so now you know what precautions to take when using the Tor Network. Essentially; only use the Tor Browser (do not route your internet traffic through Tor), make sure you only download Tor Browser from torproject.org, and try to only access Tor Hidden Services a.k.a Onion Sites rather than sites hosted on the clearnet. This gives you a great amount of anonymity and online privacy, and ensures that the government or any other malicious Tor Nodes cannot see or who you are or what you're doing. As long as you take the necessary precautions which we just talked about, you will be nearly fully anonymous while using Tor Browser!
