What is a "pastebin"?
×
A pastebin is a web application aka website where you can paste and share (via a shareable link) snippets of text, usually source code or log files, for public viewing. It is popular with users of IRC and other chat programs where pasting large amounts of text directly into the chat room is considered bad etiquette. There are many different styles of pastebins on the internet. Over time many pastebins have become specialized and have features which cater to a specific audience. Pastes made to pastebins are usually only saved for a short period of time, usually anywhere from a day to a month, although WebPaste (a Swivro service) allows you to save a paste for as long as "forever".

What is "matrix protocol"?
×
Matrix is an open standard and communication protocol for real-time communication. It aims to make real-time communication work seamlessly between different service providers. Speaklink uses the Matrix protocol, specifically using the public Matrix Server. It is extremely secure and features end to end encryption.
