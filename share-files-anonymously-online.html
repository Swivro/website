Sharing files securely

Discover how anonymously share files with anyone on the internet, without prying eyes or privacy-violating features.

Do you need to share files through the internet anonymously and securely?

Sharing files anonymously, and securely can be difficult at times. Though sending files through secure messengers like Session is considered to be secure, there are more private & anonymous ways to do such a thing. Below, we'll list some ways to anonymously share files.

1. OnionShare

OnionShare is an open-source tool that lets you securely and anonymously share a file of any size. It works by starting a web server locally, on your machine, accessible as a Tor Hidden (onion) Service, with a complicated unguessable URL that you can share with the recipients to download or send files. OnionShare also allows you to set up a dropbox for anyone to send you files anonymously. Luckily, no potentially government-ran Tor Exit Nodes are involved, since files are downloaded through a Tor Hidden Service, and not through the clearnet (Exit Nodes are only utilized on Tor when connecting to a website hosted on the clearnet).

2. Magic Wormhole

Magic Wormhole is a linux package that provides a library and a command-line tool named wormhole, which makes it possible to get arbitrary-sized files and directories (or short pieces of text) from one computer to another without prying eyes or privacy-violating features. Their motto: "Get things from one computer to another, safely". Though Magic Wormhole may be secure and privacy-oriented, OnionShare is much more easy to understand, is far more user-friendly, is better for anonymity, and supports more systems, making OnionShare an overall better solution for machines with a graphical user interface.
