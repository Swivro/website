What DNS resolvers should you be using?

By Albert A.
November 2, 2021
Chances are, you're probably using your ISP's default DNS servers. That is, right after Google, the worst DNS resolver service in the world in terms of privacy. DNS-over-HTTPS, DNS-over-TLS, and DNSCrypt resolvers will not make you anonymous. Using Anonymized DNSCrypt hides only your DNS traffic from your Internet Service Provider, so the DNS provider can still see what's going on. However, using any of these protocols will prevent DNS hijacking, and make your DNS requests harder for third parties to eavesdrop on and tamper with. If you are currently using Google's DNS resolver, you should pick an alternative here.

AdGuard

AdGuard DNS is a foolproof way to block Internet ads that does not require installing any applications. It is easy to use, absolutely free, easily set up on any device, and provides you with minimal necessary functions to block ads, counters, malicious websites, and adult content.
 
AdGuard Locations:
  • Anycast (bsaed in Cyprus)

AdGuard Logging Policy:

  • They claim to have a strict no logging policy.

AdGuard DNS Protocols:

  • DNS over HTTPS
  • DNS over TLS
  • DNSCrypt

AdGuard Filtering:

  • Blocks Ads
  • Blocks Malicious Websites
  • Blocks adult content

BlahDNS

A hobby Adblock DNS project focused on online privacy, security, and anonymity, ran on pocket money and donations. For maximum privacy, none of their servers are based in the US, UK, or France, and they utilize various secure DNS Protocols.
 
BlahDNS Locations:
  • Switzerland
  • Singapore
  • Japan
  • Germany
  • Finland

BlahDNS Logging Policy:

  • The BlahDNS website says "no logs".

BlahDNS Protocols:

  • DNS over HTTPS
  • DNS over QUIC

BlahDNS Filtering:

  • Blocks Ads
  • Blocks Malicious Websites
  • Blocks trackers

1.1.1.1 (Cloudflare DNS)

Cloudflare DNS is an enterprise-grade authoritative DNS service that offers the fastest response time, unparalleled redundancy, and advanced security with built-in DDoS mitigation and DNSSEC.
 
1.1.1.1 Locations:
  • Anycast (based in the USA)

1.1.1.1 Logging Policy:

  • Cloudflare logs limited DNS query data.

1.1.1.1 Protocols:

  • DNS over HTTPS
  • DNS over TLS

1.1.1.1 Filtering:

  • Based on server choice

LibreDNS

Cloudflare DNS is an enterprise-grade authoritative DNS service that offers the fastest response time, unparalleled redundancy, and advanced security with built-in DDoS mitigation and DNSSEC.
 
LibreDNS Locations:
  • Germany

LibreDNS Logging Policy:

  • They claim that no logs are taken, and that logs are disabled for their DNS daemon.

LibreDNS Protocols:

  • DNS over HTTPS
  • DNS over TLS

LibreDNS Filtering:

  • Blocks ads
  • Blocks trackers

NextDNS

NextDNS protects you from all kinds of security threats, blocks ads and trackers on websites and in apps and provides a safe and supervised Internet for kids — on all devices and on all networks.
 
NextDNS Locations:
  • United States
  • Areas in the European Union
  • United Kingdom
  • Switzerland

NextDNS Logging Policy:

  • You can choose how long you want your logs retained for, or you can disable logging.

NextDNS Protocols:

  • DNS over HTTPS
  • DNS over TLS

NextDNS Filtering:

  • Blocks piracy websites
  • Blocks mature content
  • You are also able to customize what you want blocked

Quad9

Quad9 is a free service that replaces your default ISP or enterprise Domain Name Server (DNS) configuration. When your computer performs any Internet transaction that uses the DNS (and most transactions do), Quad9 blocks lookups of malicious host names from an up-to-the-minute list of threats.
 
Quad9 Locations:
  • Anycast (based in Switzerland)

Quad9 Logging Policy:

  • Quad9 claims to keep limited log data with no personal info retained.

Quad9 Protocols:

  • DNS over HTTPS
  • DNS over TLS
  • DNSCrypt

Quad9 Filtering:

  • Quad9 blocks malicious domains.

 

You should carefully review the descriptions provided of the 6 best DNS resolver services listed above, and use that info to determine which one you think would work best for you. Each one has it's own unique features, but only one of them will fit your needs the best. We hope you found this article helpful, have a nice day!
