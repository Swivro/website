What messenger should you be using?

By Albert A.
November 1, 2021

If you are currently using an Instant Messenger like Telegram, LINE, Viber, WhatsApp, or plain SMS on your mobile device, we highly suggest picking an alternative, which is what this post is all about.

We only recommend instant messengers that utilize end-to-end encryption (E2EE), as E2EE encrypts all messages, voice, video, etc, before they are sent from your device to the recipient. E2EE will also protect the confidentiality and authenticity of the transmission as it passes through any part of the network. Any instant messengers we recommend below and completely free and are open source. 

Signal is a mobile and desktop app developed by Signal Messenger LLC, which is very similar to Telegram but features full E2EE. The app provides instant messaging, as well as voice and video calling. All communications through Signal are end-to-end encrypted unless you choose to send a message as SMS. The Signal Protocol was also independently audited.

Signal Benefits:

  • Cross-platform, supporting Desktop and Mobile.
  • New features and changes can be implemented more quickly.
  • Easier to get started with and to find contacts.
  • Recommended by Elon Musk :)

Signal Downsides:

  • Requires a phone number
  • Poor/No documentation for 3rd party developers
  • The ownership, privacy policy, and operations of the service can change easily when a single entity controls it, potentially compromising the service later on.
  • Centralized services could be more susceptible to legislation requiring backdoor access.

 

Element is the reference client for the Matrix network. The Matrix open standard is an open-source standard for secure (audit), federated, real-time communication. Element/Matrix utilizes end-to-end encryption, does not require a phone number, can be used in a web browser, and is completely free, having a similar feel to the popular Discord App. Federated (aka decentralized) messengers use multiple, independent servers that are able to talk to each other (email is one example of a federated service). Federation allows system administrators to control their own server and still be a part of the larger communications network. If you're a high threat level, we suggest using Element over the Tor Network in conjunction with a VPN. We suggest to either using the Public Matrix Server on Element, or create your own for maximum privacy.

Element Benefits:

  • Allows for greater control over your own data when running your own server.
  • Often allows for third party clients which can provide a more native, customized, simpler, or more accessible experience.
  • Allows you to choose who to trust your data with by choosing between multiple "public" matrix servers. We highly suggest running your own private server.
  • Generally a less juicy target for governments wanting backdoor access to everything as the trust is decentralized.
  • Server software can be verified that it matches public source code, assuming you have access to the server or you trust the person who does (ex: a family member or friend)
  • Third-party developers can contribute code and add new features, instead of waiting for a private development team to do so.

Element Downsides:

  • Adding new features is more complex, because these features need to be standardized and tested to ensure they work with all servers on the network.
  • Some metadata may be available (e.g., information like "who is talking to whom," but not actual message content if E2EE is used).
  • Federated servers generally require trusting your server's administrator. They may be a hobbyist or otherwise not a "security professional," and may not serve standard documents like a privacy policy or terms of service detailing how your data is utilized.
  • Server administrators sometimes choose to block other servers, which are a source of unmoderated abuse or break general rules of accepted behavior. This will hinder your ability to communicate with users on those servers.

You should choose which messenger you think is a better option for you. If you're not a high threat level, and you're just looking for a more private experience when talking your friends, colleagues, or family members (whether that is via messaging, voice, or video), we suggest giving Signal a try. If you're maybe a higher threat level, want maximum anonymity, or want to share as little information as possible, we suggest going with Element/Matrix. We hope this article helped you pick a privacy-focused messenger!
